Scatterplot browser


USAGE:
  • first select (in the left-hand-side panel) an identifier for the Y-axis; (hovering the mouse over these identifiers allows quick scanning over possible choices; clicking on an identifier fixes it as the choice; this choice serves as the "anchor" for all the subsequent selections, but it can be changed, as described below);
  • then select one or more identifiers for the X-axis, by clicking on them;
  • the thin crosswise marks along the Y- and X-axes correspond to points for which only the X- or Y-coordinate is specified and available;
  • to choose a different identifier for the Y-axis, click on the previously selected one; (it is possible to choose the Y-axis identifier also for the X-axis, by holding down the shift key while clicking on the Y-axis identifier the second time);
  • changing the metric (through the righ-hand-side button-set) preserves the current selection of identifier-pairs.
