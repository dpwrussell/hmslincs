Welcome to the HMS LINCS Database

  • This database is under development, improvements are ongoing.
  • The HMS LINCS Database currently contains information on perturbagens, cells, and HMS LINCS datasets. A list of the datasets available via the HMS LINCS Database can be found on the "Data being collected" page of the HMS LINCS website. Other large and more complex HMS LINCS datasets are stored in the HMS LINCS Data Repository, which will be accessible online by the end of the summer of 2011.Â  In the future it will be possible to search for metadata in Data Repository using the Database. As of July 2011, this is not yet implemented.
  • Programmatic access to the HMS LINCS Database is detailed at the Programmatic Data Access API documentation page.
  • Metadata standards for the data in the HMS LINCS DB are currently being developed in collaboration with the LINCS centers. Updated information about these standards can be found at the LINCS project website data standards page.

A quick guide to using the HMS LINCS Database (updated April 2012):

  • To browse the HMS LINCS small molecule inhibitor collection, click on "Reagents" in the Browse section on the left menu bar. This will bring up a list of the unique HMS LINCS reagents, with one batch per reagent displayed. From this browse result, click on "Browse Reagent Wells" to see a list of all batches and wells containing the reagents. Click on an individual reagent ID in the "Facility-Salt-Batch-ID"; column to see details about that compound.
  • To search for a single or list of compounds by compound name or ID#, use the 'Find Reagents by... Compound Name" feature on the left side menu.
  • To search for a cell line or cell type, open the cells browser using the "Browse... Cells" link on the left menu. There is a search menu (expandable) at the top of the Cells page. Cells used in particular assays are listed on the relevant "Screens" pages.
  • QC information for specific small molecule compound batches is accessible through the "Attached Files" links on each Reagent page (note that not all compounds have QC information available).
  • Use "Browse Screens" to access assay protocols and data from the Mitchison lab Apoptosis and Mitosis assays.
  • Use "Browse Studies" to access protocols and data from drug-target interaction profiling assays (KINOMEscan and KiNativ assays) and data on compounds profiled using the MGH/Sanger Institute growth inhibition assay.
  • To find the target(s) of an inhibitor as reported in the literature, the information can currently be found in Study 300001, or, when viewing a single compound, by scrolling to the "Studies" section at the bottom of the page.
  • Searches fulfilling more complex criteria can be accessed via "Find Reagents by...Other criteria" on the left menu bar or initiated by clicking "Search" at the top of any Database pages (e.g. a Reagents, Libraries, Screens, or Studies display page). For any search, the number of fields searched and displayed can be increased or decreased by clicking "Columns (add/remove)" once in Search mode.
  • You can download the viewed data from any page as an Excel or SD file by clicking the "Export" button.
