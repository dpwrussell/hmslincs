Welcome to the HMS LINCS Database

  • This database is under development, improvements are ongoing.
  • The HMS LINCS Database currently contains information on experimental reagents (small molecule perturbagens, cells, and proteins) and HMS LINCS datasets. A list of the datasets available via the HMS LINCS Database can be found on the "Data being collected" page of the HMS LINCS website. Other large and more complex HMS LINCS datasets are stored in the HMS LINCS Data Repository, which is also accessible via the HMS LINCS website.  In the future it will be possible to search for all metadata in Data Repository using the HMS LINCS Database. As of November 2012, this is not fully implemented.
  • Image files of experimental data are available for the following datasets: "Tang Mitosis/Apoptosis ver.II", "Tang Proliferation/Mitosis", and "Tang Mitosis/Apoptosis ver.II (2nd)"; the images are stored and served via the HMS LINCS OMERO Server (see Resources)
  • Programmatic access to the HMS LINCS Database is detailed at the HMS LINCS Database URL Scheme and Access Guide.
  • Metadata standards for the data in the HMS LINCS DB are currently being developed in collaboration with the LINCS centers. Updated information about these standards can be found at the LINCS project website data standards page.

A quick guide to using the HMS LINCS Database (updated November 2012):

  • To search for small molecule reagents, cells, or proteins in the repository, enter a search term in the search box on this page.
  • To browse a specific entity type, select from the menu on the left.
  • Soon, this interface will support searches for lists of reagents.
  • QC information for specific small molecule compound batches is accessible through the "Attached Files" links on each small molecule detail page, in the "Batch information" section (note that not all compounds have QC information available).
  • Click on datasets to access assay protocols and data from the Mitchison Lab Apoptosis and Mitosis assays, data from drug-target interaction profiling assays (KINOMEscan and KiNativ assays) and data on compounds profiled using the MGH/Sanger Institute growth inhibition assay.
  • You can download the viewed data from any page as an Excel or CSV file by clicking the respective "Export" link.
