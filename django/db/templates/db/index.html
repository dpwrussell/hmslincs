Welcome to the HMS LINCS Database

  • This database is under development, improvements are ongoing.
  • The HMS LINCS Database currently contains information on experimental reagents (small molecule perturbagens, cells, and proteins) and HMS LINCS datasets. A list of the datasets available via the HMS LINCS Database can be found on the Data being collected page of the HMS LINCS website.
  • A table summarizing all compounds for which KINOMEscan data are available in the HMS LINCS DB can be found here.
  • Image files of experimental data are available for the following datasets: Tang Mitosis/Apoptosis ver.II (PMID: 23788527), Tang Proliferation/Mitosis, and Breast cell line dose response to target inhibition measured by high throughput microscopy (PMID: 24013279); the images are stored and served via the HMS LINCS OMERO Server (see Resources)
  • Programmatic access to the HMS LINCS Database is detailed at the HMS LINCS Database URL Scheme and Access Guide.
  • Metadata standards for the data in the HMS LINCS DB are currently being developed in collaboration with the LINCS centers. Updated information about these standards can be found at the LINCS project website data standards page.
  • Citing data from the HMS LINCS DB:
    • Published data: Datasets in the HMS LINCS DB that are published are indicated by PMID numbers in the dataset titles. The appropriate publication should be cited when the data are used.
    • Unpublished data: Procedures for citing unpublished data produced by the LINCS Consortium are in development and will be posted on the lincsproject.org website. Until this information is posted, please cite the HMS LINCS Database (http://lincs.hms.harvard.edu/db/) as the source of the data, include the relevant dataset ID number(s), and note the date that the dataset was downloaded or accessed from the HMS LINCS DB for use.

A quick guide to using the HMS LINCS Database (updated Nov. 2013):

  • For detailed instructions, see the HMS LINCS DB Tutorial.
  • To find datasets from LINCS publications, type the relevant PMID in the "search all categories" search box, or the search box on the datasets page.
  • To search for small molecule reagents, cells, proteins, or specific datasets in the repository, enter a search term in the search box on this page.
  • Note that all Dataset entries have multiple tabs describing the Details (experimental and analysis protocols), Reagents Studied (used to create the dataset), Data Column headings used to format the results, and the Results.
  • A list of all Datasets in HMS LINCS DB can be obtained by clicking on the Datasets menu item on the left side of this screen. The resulting list of HMS LINCS datasets appears initially sorted by Dataset ID#. It can be re-sorted by BioAssay Type by clicking on the "BioAssay" column heading in the table.
  • To filter the results of searches: select the down arrow icon at the right side of the search box: use the expanded section to select items on which to filter your search.
  • Searching within the results of a dataset: select the “filter results” option on the results tab, then select the appropriate items (e.g. small molecule name(s), cell line name(s), etc.).
  • Select Structure Search to perform a chemical structure search for a small molecule reagent. The search will accept input in the format of SMILES or sdf file format.
  • To browse a specific entity type, select from the menu on the left.
  • Soon, this interface will support searches for lists of reagents.
  • QC information for specific small molecule compound batches is accessible through the "Attached Files" links on each small molecule detail page, in the "Batch information" section (note that not all compounds have QC information available).
  • You can download the viewed data from any page as an Excel or CSV file by clicking the respective "Export" link. Please note that the date and time of the download is included in the filename.
  • For questions or feedback, please contact: lincs-feedback@hms.harvard.edu.
